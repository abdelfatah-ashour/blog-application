# Blog Application

## Stack

React.js | Redux/toolkit | TailwindCSS | Styled-component

<br />

## How to run the application

after clone this repository and in thr root run the following command.

```bash
  npm install
```

<br />
<br />

### For development mode

```bash
  npm run dev
```

<br />

### For production mode

```bash
  npm run build
```

```bash
  npm run preview
```
